Bem vindo! Desenvolvido a carter de ESTUDO!

Esse sistema foi desenvolvido para fins educacionais, com o intuito de demonstrar a construção de um sistema em Angular.

O sistema é composto por um CRUD de produtos, onde é possível criar, ler, atualizar e deletar produtos.

Tecnologias utilizadas:
  • Angular
  • TypeScript
  • HTML
  • CSS
  • Material Design
API utilizada:
  • json-server
